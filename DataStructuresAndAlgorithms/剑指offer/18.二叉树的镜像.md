## 题目描述

操作给定的二叉树，将其变换为源二叉树的镜像。

## 输入描述

```
二叉树的镜像定义：源二叉树 
    	    8
    	   /  \
    	  6   10
    	 / \  / \
    	5  7 9 11
    	镜像二叉树
    	    8
    	   /  \
    	  10   6
    	 / \  / \
    	11 9 7  5
```

## 思路

```
当树不为空时，交换其左孩子和右孩子，并对他们的左孩子和右孩子也做同样的处理。
```

## 代码

```java
/**
public class TreeNode {
    int val = 0;
    TreeNode left = null;
    TreeNode right = null;

    public TreeNode(int val) {
        this.val = val;

    }

}
*/
public class Solution {
    public void Mirror(TreeNode root) {
        TreeNode temp;
       if(root != null){
            temp = root.left;
            root.left = root.right;
            root.right = temp; 
            Mirror(root.left);
            Mirror(root.right);
       }
    }
}
```
