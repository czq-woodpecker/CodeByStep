## 如何添加运算符

题目描述
>给出一个数字N，对于数字序列 1,2,3 ... N。现在在其中插入“+”, "-", "  "，使得表达式的和为M。" "的含义是把相邻的两个数字组成一个数。例如：1 + 2 3 - 4，含义是：1 + 23 - 4 = 20。
> 给出N和M，求出所有合法的序列的个数。

输入
> 两个整数N,M ( N <= 7, M <= 100)

输出
> 合法序列的个数


样例输入
> 7　0
> 
样例输出
> 6


分析：问题规模不大n<=7，m<=100，直接dfs暴力求解，这里用到了eval()，也可以自己实现转化为后缀表达式求值的函数；

## AC代码
```python
n,m = map(int, raw_input().split())

res = [0]

def dfs(i,s):
  if i == n:
      if m == eval(s):
          res[0] += 1
  else:
      dfs(i+1,s+'+'+str(i+1))
      dfs(i+1,s+'-'+str(i+1))
      dfs(i+1,s+str(i+1))

dfs(1,'1')
print res[0]
```
